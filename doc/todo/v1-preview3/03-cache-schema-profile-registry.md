# 03. Cache, Schema, and Profile Registry

- [x] Cache keys are created through `@nnrp/core`.
- [x] Schema descriptors are created through `@nnrp/core`.
- [x] Standard input profiles are exported through `@nnrp/core`.
- [x] Native and browser role packages advertise cache, schema, and result-hint capabilities through their manifests.
- [x] Public examples use role package entrypoints rather than internal artifact paths.
