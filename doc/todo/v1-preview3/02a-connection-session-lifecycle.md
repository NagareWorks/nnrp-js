# 02a. Connection and Session Lifecycle

- [x] Native client lifecycle is exposed from `@nnrp/native-client`.
- [x] Native server lifecycle is exposed from `@nnrp/native-server`.
- [x] Browser client lifecycle is exposed from `@nnrp/browser-client`.
- [x] Role packages keep client-only and server-only exports separated.
- [x] Runtime close paths are idempotent and reject operations after close.
- [x] Session metadata is validated before dispatch.
